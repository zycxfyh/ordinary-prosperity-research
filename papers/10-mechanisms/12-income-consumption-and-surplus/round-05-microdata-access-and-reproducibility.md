# 从公开资料到可复现微观估计：数据访问、分析单位与原型 Y 提取协议

> Issue #1 · 第五轮  
> 阶段：微观数据访问审计、分析单位修正与可复现提取设计  
> 日期：2026-07-20

## 摘要

第四轮建立了25—34岁青年原型的公开统计锚点，但仍未形成美国、中国和澳大利亚同口径的微观估计。第五轮因此转向三个问题：数据是否可取得、分析单位是否一致、取得后应如何构造可复现样本。

本轮首先修正了原型定义中的一个逻辑冲突：研究焦点个人、住宅中的共同居住家庭与实际共享收入支出的经济单位不是同一对象。一个青年可以与父母同住，但保持部分财务独立；也可以住在同一住宅并与父母共享主要支出。美国 Consumer Expenditure Survey 使用 consumer unit，强调共同承担主要支出；CHFS以家庭和个人文件记录住宅家庭及成员；HILDA同时提供个人与家庭层文件。若不区分这三层对象，共同居住、个人收入和家庭盈余就会被错误拼接。

数据访问审计显示：美国CE 2024 public-use microdata可公开下载，提供CSV、Stata和SAS格式及官方R示例代码；但2024年不再提供联邦税、州税和税后收入估计，因此严格的现金可支配收入分析应以2023年作为主基准，或自行运行并公开税收估计模型。CHFS 2021已向实名注册用户开放，且2026年7月完成2011—2021六轮数据标准化更新，明确提供家庭问卷、个人问卷、家庭综合和个人综合四类文件。HILDA Release 24向获批研究人员开放，需机构邮箱、Dataverse申请和保密承诺，官方文档同时提供权重、文件匹配、纵向构造和税收福利模型程序。

本轮尚未报告新的三国微观估计。原因不是研究设计缺失，而是CHFS与HILDA需要由用户以本人和机构身份申请；美国BLS二进制数据虽公开，但本轮运行环境未能直接摄取ZIP/XLSX文件。第五轮的成果是把下一次执行需要的数据、单位、筛选条件、权重、年份与误差估计方法固定下来，使取得文件后能够直接计算，而不是重新讨论概念。

## 1. 三层分析单位

### 1.1 焦点个人

焦点个人是本研究希望描述的25—34岁青年。个人层变量包括：

- 年龄、性别、教育和就业；
- 个人劳动收入与工时；
- 个人养老金、债务和流动资产；
- 与伴侣、父母和子女的关系；
- 个人对账户和资产的控制。

### 1.2 住宅家庭

住宅家庭指同一住所中的全部成员。它用于观察：

- 是否与父母或其他成年人共同居住；
- 成年人数和儿童人数；
- 住房产权与按揭；
- 谁拥有住房；
- 照护和实物支持。

住宅家庭成员不一定完全共享收入和支出。

### 1.3 经济共享单位

经济共享单位是实际共同承担主要生活支出、汇集资源或作出共同财务决定的人群。它决定：

- 应把哪些收入加总；
- 哪些住房、食品和交通支出属于同一预算；
- 家庭支持是内部资源，还是外部转移；
- 盈余究竟属于个人还是共同单位。

美国CE的consumer unit可能是一个独居者，也可能是与他人同住但财务独立的单人单位；还可能是共同承担主要支出的多成员单位。因此“一人consumer unit”不能自动解释为“住宅中只有一个人”。[@bls2026glossary]

## 2. 修正后的原型 Y

第五轮把原型Y拆成三个相互排斥的状态。

### Y1：独立单人经济单位

- 焦点个人25—34岁；
- 经济共享单位只有一人；
- 无伴侣和共同抚养子女；
- 可以独居，也可以与他人同住但不共享主要支出；
- 住房状态进一步区分租房、有按揭和无按揭。

### Y2：与父母共同居住、部分财务独立

- 焦点个人25—34岁；
- 与至少一名父母同住；
- 个人拥有劳动收入；
- 部分支出自行承担，部分住房或生活服务由父母提供；
- 必须估计住房实物转移、board、现金支持与反向转移。

### Y3：与父母共同居住、资源高度共享

- 焦点个人25—34岁；
- 与父母属于同一经济共享单位；
- 家庭收入和支出共同决定；
- 不能用个人工资减个人消费表示盈余；
- 应同时报告家庭盈余和焦点个人的经济自主指标。

因此，“共同居住”不再作为单人原型的一种简单住房标签，而成为需要单独识别资源共享程度的家庭制度。

## 3. 美国CE微观数据方案

### 3.1 可用数据

BLS已发布2024年CE Public Use Microdata。数据以SAS、Stata和CSV三种格式提供，Interview和Diary Survey分别打包；2024年CSV Interview包约52.4MB。PUMD允许研究者分析公开表无法同时交叉的收入、支出与人口特征。[@bls2026pumddata; @bls2026pumdguide]

Interview Survey主要文件包括：

- `FMLI`：consumer unit层面的汇总支出、收入、资产负债特征和权重；
- `MEMI`：成员层人口与收入特征；
- `ITBI`：详细收入；
- `MTBI`：按UCC编码的月度支出；
- 其他住房、车辆、保险和债务详细文件。

文件通过`NEWID`关联。BLS提醒，FMLI上的收入汇总变量与正式表格定义可能存在差异；重建收入时需要检查ITBI和UCC口径，而不能把单个汇总变量直接当作官方年度收入。[@bls2026pumdguide; @bls2026faq]

### 3.2 年份选择

主估计采用2023年，2024年作为支出更新与敏感性分析。

原因是BLS所使用的外部税收估计模型没有更新到2024税年，因此2024年的CE表、LABSTAT和PUMD均不再提供联邦税、州税和税后收入。BLS仍提供tax-unit identifier，研究者可以自行构造税收模型输入。[@bls2026aftertax; @bls2026faq]

由此形成两套结果：

1. **2023完整现金预算**：使用BLS TAXSIM35税后收入；
2. **2024当前支出结构**：报告税前收入和支出，税后结果仅在独立税收模型完成后发布。

### 3.3 日历年拼接

自2020年起，一个年度Interview包只包含当年第二至第四季度和下一年第一季度。构造完整日历年时，还需要从前一年包取得当年第一季度，并根据访谈月份处理实际属于目标年度的月份。[@bls2026pumddata; @bls2026pumdguide]

2023年估计需要：

```text
2022 package中的2023 Q1
+ 2023 package中的2023 Q2-Q4
+ 2023 package中的2024 Q1所回溯的2023月份
```

2024年估计需要：

```text
2023 package中的2024 Q1
+ 2024 package中的2024 Q2-Q4
+ 2024 package中的2025 Q1所回溯的2024月份
```

### 3.4 样本筛选

第一版美国Y1样本条件：

```text
25 <= focal_age <= 34
AND economic_unit_size == 1
AND earner_count >= 1
AND no_co_resident_partner_in_economic_unit
AND no_dependent_child_in_economic_unit
```

再按住房状态分层：

```text
renter
owner_with_mortgage
owner_without_mortgage
```

该筛选产生的是“单人经济单位”，不是严格意义的独居人口。住宅共同居住需要使用成员关系、住房付款和其他调查补充资料单独识别。

### 3.5 指标构造

每组至少报告：

- 税前收入；
- 税后现金收入；
- 工资薪金；
- 住房、食品、交通、医疗与债务付款；
- 养老金和Social Security；
- 按揭本金；
- 现金盈余；
- 广义资本形成；
- 流动资产；
- 住房权益；
- 家庭外部现金支持。

现金盈余定义为：

```text
税后现金收入
- 经常性消费
- 住房利息与持有成本
- 必要债务付款
- 当前现金缴费
```

广义资本形成在现金盈余基础上加入个人退休缴费和按揭本金，但Social Security缴费与未来权益应单独报告，不能简单等同于个人账户资产。

### 3.6 权重与不确定性

BLS要求使用`FINLWT21`形成总体代表性估计，并按季度数与在目标年份范围内的月份进行调整。标准误应使用`WTREP01`至`WTREP44`进行Balanced Repeated Replication。BLS明确指出，这一普通BRR方法不适用于插补收入的标准误，因此收入不确定性必须依照专门的收入插补方法处理。[@bls2026pumdguide]

结果至少报告：

- 未加权样本数；
- 加权总体规模；
- 加权均值；
- 加权中位数和分位数；
- BRR标准误或置信区间；
- 零值比例；
- 插补、分配和top-code比例。

## 4. 中国CHFS微观数据方案

### 4.1 访问状态

CHFS 2021已开放，用户需要在官方数据网站实名注册、申请并通过审核后下载。2021年样本为22,027户，覆盖29个省级地区和269个区县，约70%家庭在2021年调查，剩余约30%在2022年完成。[@chfs2026public2021]

2026年7月，CHFS完成2011—2021六轮公开数据标准化更新。每轮明确设置四类文件：

- `hh`：家庭问卷；
- `ind`：个人问卷；
- `master_hh`：家庭编码、家庭权重、地理标识及收入、消费、资产和负债综合变量；
- `master_ind`：家庭与个人编码、个人权重、工资和养老金等综合变量。[@chfs2026update]

第五轮后续分析必须使用这次标准化后的版本，而不是此前下载或论文附件中的旧数据。

### 4.2 构造路径

CHFS不应先筛选“户主25—34岁”，而应先在个人文件中识别所有25—34岁的焦点个人，再关联其住宅家庭。

核心步骤：

1. 从`master_ind`筛选25—34岁并识别就业、工资与个人权重；
2. 使用家庭编码关联`ind`，取得父母、伴侣和子女关系；
3. 使用家庭编码关联`master_hh`，取得家庭收入、消费、资产、负债和家庭权重；
4. 使用`hh`核对住房数量、产权、按揭、租金、家庭转移和支出细项；
5. 根据居住成员与资源共享情况分类Y1、Y2、Y3；
6. 同时保留个人指标和家庭指标。

### 4.3 需要避免的错误

- 不能把全国人均可支配收入赋给焦点青年；
- 不能把家庭收入全部视为焦点个人可控制资源；
- 不能把与父母同住者的低住房现金支出解释为独立生活成本低；
- 不能把多名25—34岁成员的同一家庭重复计入家庭总体估计；
- 必须区分个人权重和家庭权重；
- 2021与2022调查月份需记录，并在价格处理时统一。

## 5. 澳大利亚HILDA微观数据方案

### 5.1 访问状态

HILDA Release 24包含累计波次数据，研究人员需通过DSS Longitudinal Studies Dataverse申请。数据免费，但仅向获批的政府、学术和非营利机构研究人员开放；申请通常要求机构邮箱、在线申请和签署Confidentiality Deed Poll。General Release可向澳大利亚及海外研究人员开放，Restricted Release只向有充分理由的澳大利亚用户提供。[@hilda2026datausers; @dss2024dataaccess]

### 5.2 文档与可复现支持

HILDA User Manual覆盖：

- 缺失值规则；
- 派生变量；
- 家庭和个人文件匹配；
- 收入与支出插补；
- 权重；
- 数据质量。

官方Program Library还提供R、Stata、SAS和SPSS示例，用于家庭—个人文件匹配、创建纵向文件、应用权重以及运行HILDA税收福利模型。[@hilda2026manual; @hilda2026programlibrary]

### 5.3 构造路径

HILDA没有预先定义唯一的“家庭参考人”，研究者应围绕焦点个人构造样本，而不是依赖户主概念。个人跨波次由`xwaveid`关联；同波次家庭成员共享家庭标识，伴侣可以通过官方partner identifier关联。[@hilda2026faq]

第一版使用Wave 24横截面：

1. 识别25—34岁受访者；
2. 筛选有劳动收入或就业者；
3. 排除同住伴侣和受抚养子女，构造Y1；
4. 识别与父母同住者并构造Y2/Y3；
5. 关联家庭住房、租金、按揭、财富和家庭收入；
6. 关联个人收入、税后收入、Super、个人债务和家庭转移；
7. 使用横截面响应者权重；
8. 后续使用多波次数据观察从共同居住到租房或购房的转移。

澳大利亚结果应分别报告：

```text
当前现金收入
+ 家庭现金与实物支持
+ 雇主Super权益
+ 住房本金形成
```

否则会在只看银行现金时低估长期资本，在把Super全部视为当前可用资产时高估短期选择权。

## 6. 三国共同输出协议

最终三国原型表不直接排列货币金额，而先在各国内部计算：

| 维度 | 主要指标 |
|---|---|
| 收入 | 个人劳动收入、经济单位市场收入、税后现金收入 |
| 家庭支持 | 现金转移、住房实物服务、board与反向转移 |
| 固定成本 | 住房、交通、债务、保险和必要缴费 |
| 盈余 | 现金盈余中位数、负盈余比例、盈余分位数 |
| 资本形成 | 退休缴费、Super、按揭本金、金融资产购买 |
| 流动性 | 现金和可迅速出售资产、必要支出覆盖月数 |
| 锁定资本 | 住房权益、养老金和企业资产 |
| 选择权 | 收入中断承受月数、迁移与换岗约束 |

跨国比较采用三种方式：

1. 占税后收入比例；
2. 必要支出覆盖月数；
3. PPP调整的消费能力和住房负担。

任何一张表都必须同时标记：年份、调查单位、样本筛选、权重、是否插补、是否top-code、是否包含公共实物转移以及养老金和按揭本金处理。

## 7. 本轮执行结果与限制

本轮已验证：

- BLS 2024 PUMD及2023—2024交叉表的官方存在与下载路径；
- 2024税后收入不可用这一方法变化；
- CHFS 2021申请入口及2026标准化文件结构；
- HILDA Release 24申请条件、文档和官方程序支持。

本轮尚未完成：

- 下载并运行BLS单位记录；
- 以本人身份申请CHFS；
- 以机构身份申请HILDA；
- 产生三国加权中位数、分位数和标准误。

其中CHFS和HILDA访问涉及用户身份、机构邮箱和保密承诺，不能由自动化代理代签或绕过。BLS数据是公开数据，但本轮运行环境的网页层只能解析文档页面，无法摄取ZIP和XLSX二进制文件；这属于执行环境限制，不是数据不可得。

## 8. 第五轮结论

第五轮没有得到新的国家排名，但使研究向前推进了三步：

1. 修正了“个人—住宅家庭—经济共享单位”的本体混淆；
2. 确定了2023而非2024作为美国严格税后现金预算的主年份；
3. 把三国数据获取、文件关联、样本筛选、权重和误差估计转化为可直接执行的协议。

下一次取得数据后，应首先完成美国CE 2023的Y1租户与按揭业主估计；随后使用标准化后的CHFS 2021和HILDA Release 24构造Y1、Y2和Y3。Issue #1只有在这些微观结果完成并通过口径复核后，才进入机制论文定稿阶段。
