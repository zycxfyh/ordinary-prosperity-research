# 跨国比较设计

## 1. 比较目的

跨国比较的目的不是给国家排出单一优劣顺序，而是识别：

1. 哪些经济富足机制在不同社会中普遍存在；
2. 哪些机制依赖特定制度安排；
3. 同一私人行为为何在不同国家产生不同结果；
4. 私人资产、家庭支持与公共制度如何相互替代或互补。

## 2. 三维比较框架

本研究使用三个相互交叉的维度。

### 2.1 问题与机制

论文的主要组织轴包括：

- 人力资本与劳动收入；
- 收入、消费与可支配盈余；
- 储蓄、养老金与资产所有；
- 住房与债务；
- 家庭分工与代际转移；
- 社会保险与公共制度；
- 风险、韧性与经济选择权。

### 2.2 国家与制度环境

国家用于提供制度情境。首轮案例为：

- 美国；
- 中国；
- 澳大利亚。

选择三国的原因在于其教育融资、劳动市场、住房、医疗、养老金、家庭责任和移民制度具有明显差异。

### 2.3 生命历程

生命历程按社会经济转折而非固定年龄组织：

| 阶段 | 主要转折 |
|---|---|
| 进入期 | 教育完成、进入劳动市场、债务与资产起点形成 |
| 建立期 | 职业资本形成、收入增长、初始储蓄与资产参与 |
| 承诺期 | 住房、婚姻、生育、迁移与长期债务形成 |
| 积累期 | 收入高峰、资产扩张、子女教育与父母照护 |
| 转化期 | 退休准备、劳动收入向资产与制度收入转化 |
| 提取期 | 养老金、资产提款、健康与长期照护 |

具体年龄将在每个国家内部依据教育、就业、家庭与退休制度确定。

## 3. 比较矩阵

每项机制研究应尽可能填充以下矩阵：

| 机制 | 美国 | 中国 | 澳大利亚 | 主要可比性问题 |
|---|---|---|---|---|
| 人力资本与收入 |  |  |  | 教育口径、工资单位、劳动参与 |
| 生活成本与盈余 |  |  |  | 家庭规模、住房、医疗、税后口径 |
| 储蓄与金融资产 |  |  |  | 账户制度、养老金、金融参与 |
| 住房与债务 |  |  |  | 自有率、按揭、房价、租赁制度 |
| 家庭与代际支持 |  |  |  | 家庭定义、转移支付、照护劳动 |
| 社会保险 |  |  |  | 资格、覆盖、待遇与私人替代 |
| 风险与韧性 |  |  |  | 失业、医疗、流动性与恢复指标 |

## 4. 分析单位

每项比较必须明确使用的单位：

- 个人；
- 税收单位；
- 家庭；
- 住户；
- 等价化家庭成员；
- 劳动者或特定年龄群体。

跨国比较优先采用可协调的个人或家庭指标。不能协调时，应保留原始制度口径，并明确解释差异。

## 5. 核心比较指标

### 5.1 收入与劳动

- 税前与税后收入中位数；
- 工资增长轨迹；
- 劳动参与与失业；
- 教育与职业收入差异；
- 工作时间与非标准就业；
- 收入波动性。

### 5.2 消费与盈余

- 必要生活支出；
- 住房、交通、医疗和教育负担；
- 家庭储蓄率；
- 固定承诺占可靠收入比例。

### 5.3 资产与债务

- 净财富中位数；
- 流动金融资产；
- 住房权益；
- 养老金权益；
- 消费、教育与住房债务；
- 资产参与率和集中度。

### 5.4 韧性与选择权

- 应急支出承受能力；
- 收入中断覆盖期；
- 社会保险覆盖；
- 职业、地域与家庭退出能力的代理指标；
- 高固定负担和高相关性暴露。

## 6. 共同路径与国家路径

比较结论将分为两类。

### 6.1 共同机制

例如：

- 劳动收入必须形成持续盈余，才可能转化为资产；
- 高成本债务和流动性不足会中断长期积累；
- 早期人力资本对长期结果具有累积作用；
- 家庭结构既能分担风险，也能集中风险。

### 6.2 制度特定机制

例如：

- 教育成本由个人、家庭或公共部门承担的比例；
- 医疗风险是否需要大量私人储备；
- 退休积累主要依赖公共养老金、强制账户或自愿储蓄；
- 住房在家庭资产负债表中的中心程度；
- 移民身份对劳动与社会权利的影响。

## 7. 国别研究的统一提问

每个国家报告应回答：

1. 普通人的典型经济起点是什么？
2. 劳动收入如何增长并受到何种风险影响？
3. 哪些支出最显著地压缩可支配盈余？
4. 普通人通过哪些渠道持有资产？
5. 家庭和公共制度分别承担哪些风险？
6. 哪些人生转折最显著地改变长期路径？
7. 哪些可控决定具有高影响？
8. 哪些常见观念与现有证据不一致？

## 8. 比较输出

跨国比较最终应形成：

- 指标对照表；
- 制度机制图；
- 生命周期路径对照；
- 共同机制与制度条件清单；
- 数据口径与不可比性说明；
- 可检验的后续研究命题。
