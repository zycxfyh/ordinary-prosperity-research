# 普通人的经济富足

**Ordinary Prosperity Research**

本仓库研究一个基础而长期的问题：

> 在现代社会中，一个主要依靠劳动收入、缺乏巨额继承与异常创业收益的普通人，如何逐步获得经济稳定、韧性、自主与更大的生活选择权？

研究关注劳动、人力资本、收入、消费、储蓄、资产所有、住房、家庭、社会保障与制度环境之间的连接。核心分析链条是：

```text
劳动与人力资本
→ 劳动收入
→ 可支配盈余
→ 资产所有
→ 风险吸收能力
→ 经济选择权
```

## 研究组织

本仓库采用论文型结构：

- **问题与机制**构成主要研究主轴；
- **国家**提供制度情境与比较案例；
- **生命历程**用于分析进入劳动市场、家庭形成、资产积累与退休转化等关键转折；
- **证据底稿**与论文论证分离管理，以便跨论文复用和口径核对。

首轮研究将以美国、中国和澳大利亚为主要比较对象，并逐步形成：

1. 理论与方法论文；
2. 人力资本、盈余形成、资产所有、住房、家庭、社会保险与经济韧性等机制论文；
3. 国别研究；
4. 跨国综合研究。


## 研究原则

- 区分收入、财富、消费能力、经济安全与经济选择权；
- 同时观察个人与家庭两个分析层级；
- 区分平均数、中位数与分布；
- 区分存量、流量、名义值与实际值；
- 区分描述性事实、因果证据、理论解释与价值判断；
- 将跨国共同机制与制度特定机制分别陈述；
- 将不确定性、数据口径和替代解释纳入正文论证。

English working title:

> **Ordinary Prosperity: The Formation of Income, Capital, and Economic Optionality in Modern Societies**
