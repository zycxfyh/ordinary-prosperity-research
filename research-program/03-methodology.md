# 研究方法

## 1. 方法概述

本项目采用跨学科、比较性和生命历程导向的研究方法。主要结合：

- 描述性统计；
- 制度比较；
- 文献综合；
- 生命周期分析；
- 机制分析；
- 情景计算；
- 在条件允许时使用因果证据。

研究目标不是用单一模型解释所有国家和个体，而是在统一概念与可比较指标的基础上，识别普通人经济富足形成中的主要机制、制度条件与异质性。

## 2. 证据类型

每项重要陈述应尽可能标注其证据性质。

| 类型 | 含义 |
|---|---|
| 描述性事实 | 对人口、收入、支出、资产或制度状态的观察 |
| 统计估计 | 基于样本或模型得到的总体估计 |
| 因果证据 | 具有明确识别策略、能够支持因果解释的研究 |
| 制度事实 | 当前法律、政策、资格、税率或制度安排 |
| 理论解释 | 基于经济或社会机制对事实进行解释 |
| 研究命题 | 需要进一步证据检验的假设 |
| 价值判断 | 关于何种生活或分配结果更值得追求的规范性判断 |

论文应避免把描述性相关直接陈述为因果关系，也应避免把理论上可能成立的机制写成已经得到经验确认的事实。

## 3. 来源优先级

证据搜集优先考虑：

1. 国家统计机构与行政数据；
2. 税务、教育、养老金、医疗、劳动和金融监管机构；
3. OECD、ILO、世界银行、IMF等国际机构；
4. 同行评审论文；
5. 具有透明方法的高质量工作论文和研究项目；
6. 可靠研究机构和调查；
7. 高质量新闻，用于记录近期制度或社会变化。

商业内容、媒体评论和社交平台可用于发现问题，但不应单独支撑核心结论。

## 4. 数据口径

### 4.1 平均数、中位数与分布

收入和财富通常具有偏斜分布。研究应优先同时报告：

- 中位数；
- 平均数；
- 分位数或群体分布；
- 样本规模和覆盖范围。

平均数适合总量与宏观资源分析，中位数更接近典型个体状态，分布用于识别阶层和异质性。

### 4.2 存量与流量

- 收入、消费、储蓄和税收属于期间流量；
- 资产、债务、养老金权益和净财富属于时点存量。

论文不得直接比较不同时间基准或将流量与存量混为同一类指标。

### 4.3 名义值与实际值

跨时间分析优先使用经通胀调整的实际值，并明确基准年份。跨国比较应说明：

- 市场汇率；
- 购买力平价；
- 本币实际值；
- 各自适用的研究问题。

### 4.4 税前与税后

研究收入时应区分：

- 市场收入；
- 税前总收入；
- 税后可支配收入；
- 包含实物公共服务后的扩展收入概念。

不同国家私人支出需求受公共服务影响，单纯比较税前工资可能产生误导。

## 5. 分析单位

### 5.1 个人

适合分析：

- 教育与技能；
- 工资与职业；
- 劳动参与；
- 法律和社会权利；
- 经济自主与退出能力。

### 5.2 家庭或住户

适合分析：

- 收入汇集；
- 消费和储蓄；
- 住房与债务；
- 子女和照护；
- 共同资产负债表。

### 5.3 等价化处理

比较不同家庭规模时，应考虑成人和儿童数量、规模经济及共享消费。任何等价尺度都具有假设，应记录所用公式和替代口径。

## 6. 生命周期与队列

研究将区分：

- 年龄效应；
- 历史时期效应；
- 出生队列效应；
- 社会经济转折。

横截面中老年人比年轻人更富有，并不能单独证明同一年轻人在未来会复制当前老年人的路径。住房价格、养老金制度、教育成本和劳动市场可能已经发生结构变化。

因此，研究应在可能时结合：

- 横截面调查；
- 纵向追踪数据；
- 出生队列比较；
- 历史制度变化。

## 7. 比较方法

跨国比较分三步进行：

1. **概念协调**：确认各国指标是否测量同一现象；
2. **口径记录**：保存单位、年份、人口、家庭定义与数据来源；
3. **机制解释**：分析差异来自行为、市场、家庭还是制度。

当指标不可直接协调时，研究将保留国家原始口径，并通过并列解释而非强行合并完成比较。

## 8. 因果推断

项目将优先吸收具有可信识别策略的研究，例如：

- 自然实验；
- 政策变化；
- 随机试验；
- 回归不连续；
- 工具变量；
- 双重差分；
- 高质量纵向研究。

但因果研究通常针对特定人群、时期和制度。其外部有效性需要单独讨论，不能直接推广为全球一般规律。

## 9. 情景计算

情景计算用于展示数量级、敏感性和机制，不用于宣称精确预测。每个情景应明确：

- 起点；
- 时间范围；
- 名义或实际口径；
- 收入和支出假设；
- 回报与风险假设；
- 未纳入的制度或家庭因素；
- 对结论影响最大的参数。

## 10. 引用与证据记录

所有进入论文的重要来源应记录在：

- `evidence/source-register.csv`；
- `bibliography/master.bib`；
- 对应论文的参考文献文件。

证据记录至少包括：来源、标题、年份、访问日期、国家、主题、指标、人口范围、主要结论和可比性备注。

## 11. 修订原则

研究结论应随新证据修订。修订时应区分：

- 数据更新；
- 制度变化；
- 概念修正；
- 新的因果证据；
- 原有解释被替代；
- 价值判断发生变化。

论文版本应保留主要结论变化及其证据原因。
