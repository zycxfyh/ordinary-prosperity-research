# 概念框架

## 1. 研究对象

本研究将经济富足理解为一种逐步形成的能力状态，而不是单一收入水平或净资产门槛。一个人是否经济富足，取决于其能否维持合理生活、吸收常见冲击、保留长期选择，并降低对单一收入来源、关系或机构的依赖。

## 2. 五个核心概念

### 2.1 劳动收入

劳动收入是普通人最主要的初始资本来源。它受到教育、技能、职业、行业、地域、制度和家庭条件共同影响。

### 2.2 可支配盈余

可支配盈余是税后收入扣除维持当前生活与既有义务所需支出后的剩余资源。它决定劳动收入能否被转化为未来资本。

### 2.3 资产所有

资产所有包括流动储备、金融资产、住房权益、养老金权益与企业权益。不同资产在收益、流动性、风险、制度保护和选择权方面具有不同作用。

### 2.4 经济韧性

经济韧性是个人或家庭抵抗、应对并从收入、健康、家庭或市场冲击中恢复的能力。韧性既来自私人资产，也来自保险、社会保障、家庭支持与可迁移的人力资本。

### 2.5 经济选择权

经济选择权是个人能够在不立即危及基本生活的情况下，拒绝、推迟或改变工作、居住、教育、家庭和时间安排的能力。

经济选择权不等同于财富最大化。它强调资源如何转化为现实可行的替代方案。

## 3. 资本转换链

普通人的长期经济路径可表示为：

```text
初始条件
→ 人力资本形成
→ 劳动收入
→ 可支配盈余
→ 资产所有
→ 风险吸收能力
→ 经济选择权
```

每个转换环节都可能出现损耗：

- 人力资本未能转化为市场收入；
- 收入被高生活成本与固定承诺吸收；
- 盈余未能转化为生产性或保值资产；
- 资产过度集中、缺乏流动性或被高债务抵消；
- 冲击迫使家庭在不利条件下出售资产；
- 家庭财富存在，但个体成员缺乏控制权和退出能力。

## 4. 多层经济状态

本研究暂将普通人的经济状态分为五层：

| 层级 | 状态 | 核心特征 |
|---|---|---|
| 1 | 生存 | 基本生活高度依赖当期收入，普通冲击即可造成中断 |
| 2 | 稳定 | 经常性收入覆盖经常性支出，债务与账单可持续 |
| 3 | 韧性 | 拥有流动性、保险或制度支持，能够吸收常见冲击 |
| 4 | 自主 | 能够改变工作、学习、居住或家庭安排而不立即失稳 |
| 5 | 富足 | 资产、收入与制度权益持续支持广泛而稳定的选择 |

这些层级用于组织研究问题，而非建立统一的全球财富排名。

## 5. 个人与家庭的双层结构

个人经济状态与家庭经济状态必须分别观察。

### 个人层级

- 人力资本；
- 个人收入；
- 个人资产与养老金；
- 法律和社会权利；
- 职业与地域流动能力；
- 独立退出能力。

### 家庭层级

- 汇集收入；
- 共同消费；
- 住房和债务；
- 子女与照护责任；
- 共同资产负债表；
- 家庭内部风险分配。

家庭总财富增加，并不必然意味着每位成员的经济选择权同步增加。

## 6. 四类影响力量

经济富足的形成受到四类力量共同影响：

1. **市场机制**：工资、资产收益、住房价格、行业变化；
2. **制度机制**：税收、教育、医疗、养老金、劳动保护与移民规则；
3. **家庭机制**：收入共享、照护、代际转移、婚姻与家庭分工；
4. **行为机制**：现时偏好、损失厌恶、从众、过度自信与执行摩擦。

## 7. 初步理论命题

### 命题一：早期人力资本命题

在金融资产较少的青年阶段，收入能力与职业轨迹通常比合理投资组合之间的小幅回报差异更重要。

### 命题二：盈余转换命题

高收入只有在能够持续形成可支配盈余时，才会转化为长期资产和经济选择权。

### 命题三：不中断命题

长期积累的关键不仅是平均回报，还包括避免高杠杆、流动性枯竭和被迫退出。

### 命题四：制度替代命题

公共医疗、养老金、失业保障与教育支持可以替代部分私人储备需求，但也可能通过税收、资格与可携带性改变个人策略。

### 命题五：家庭非统一命题

家庭可以产生规模经济和风险共享，也可能通过不对称分工、共同债务和控制权分配削弱部分成员的经济自主。

### 命题六：选择权命题

低固定承诺、流动性和可迁移人力资本对经济选择权的贡献，可能高于相同金额但高度锁定或集中化的资产。

## 8. 后续发展

该框架将在国别研究和机制论文中接受修正。后续工作需要进一步明确：

- 经济选择权的可观察代理指标；
- 私人资产与制度权益的可比表达；
- 不同生命阶段的转换速度与中断概率；
- 国家内部阶层、地区、性别、家庭与移民身份的异质性。
