# Issue #1 第二轮可比性矩阵

> 研究对象：收入如何转化为可支配盈余  
> 比较国家：美国、中国、澳大利亚  
> 日期：2026-07-20

## 1. 矩阵用途

本文件记录三国收入、消费、储蓄、流动性与资产指标之间的可比程度。它不是结果排名表，而是后续实证分析的口径控制表。

可比性等级：

- **A：可直接比较**——概念、分析单位、时期和价格口径经过协调；
- **B：条件比较**——经家庭规模、价格、养老金或住房服务调整后可以比较；
- **C：只能解释本国内部结构**——适合本国分布分析，不能直接横向排列；
- **D：不可直接比较**——定义或统计单位差异过大，只能作为研究线索。

## 3. 分析单位映射

| 来源 | 原始分析单位 | 与本项目标准单位的关系 |
|---|---|---|
| U.S. Census CPS ASEC | household、family、person | 收入分布可用；需要明确选择household或family并做家庭规模调整 |
| BLS Consumer Expenditure Survey | consumer unit | 与household接近但不完全相同；适合消费结构，不与Census中位收入直接相减 |
| Federal Reserve SCF | primary economic unit／family | 适合家庭资产负债表；家庭定义与Census不同 |
| Federal Reserve SHED | adult respondent及其family | 适合预算余量和流动性自报；不是完整家庭账户 |
| 中国国家统计局住户调查 | resident household与常住人口人均 | 全国公开结果多为人均；需要恢复家庭规模和分布才能描述典型家庭 |
| CHFS | household与individual | 可同时分析家庭资产负债表和成员差异 |
| ABS SIH/HES | income unit、household、person | 可构造等价化家庭可支配收入；需明确household与income unit用途 |
| HILDA | household与person longitudinal | 适合生命历程和家庭转变；需处理样本流失和面板权重 |
| 国民账户 | household sector | 只表示整个家庭部门，不能替代中位家庭和群体分布 |

## 4. 收入概念映射

```text
市场收入
= 工资 + 经营收入 + 财产／资本收入 + 私人转移

现金可支配收入
= 市场收入 + 现金公共转移 - 所得税 - 社会缴费

调整后可支配收入
= 现金可支配收入 + 医疗、教育、住房等社会实物转移
```

三国公开统计并不总是按这一完整链条发布：

- 美国Census money income未包含全部非现金福利，CBO和BEA提供更广口径；
- 中国国家统计局可支配收入包括现金和实物收入，并包含自有住房折算净租金等项目；
- 澳大利亚SIH区分private、gross、disposable、adjusted与final income，最接近完整层级体系。

## 5. 消费与储蓄处理

### 5.1 养老金缴费

养老金缴费在不同资料中可能被处理为：

- 当前支出；
- 强制社会缴费；
- 家庭储蓄；
- 雇主薪酬的一部分；
- 养老金权益变化。

后续比较至少报告两个结果：

1. **可自由支配现金盈余**：不把锁定养老金视为当前现金；
2. **广义资本形成**：把养老金权益增加计入长期储蓄。

### 5.2 自有住房

需要区分：

- 按揭本金偿还：资产负债表转换，不是纯消费；
- 按揭利息：融资成本；
- 维修、税费和保险：持有成本；
- 自有住房折算租金：国民账户或部分收入统计中的推算服务流；
- 房价变化：资本利得，不是当期储蓄流量。

### 5.3 耐用品与教育

车辆、家电、教育和职业培训同时具有当前消费和未来服务／人力资本投资属性。第一轮比较沿用官方消费分类，专题研究再做重新分类。

## 6. 价格与时期协调

后续数值比较需按以下顺序处理：

1. 首先在各国内部使用同一年或相邻年份的收入和消费资料；
2. 转换为各国实际本币，注明价格基期；
3. 进行跨国生活水平比较时使用PPP，而不是市场汇率；
4. 对住房、医疗和教育另行报告实物数量与负担率，避免PPP掩盖制度结构；
5. 当前最新年份不一致时，优先保持统计质量，并明确参考期差异。

## 7. 第二轮可用结论等级

| 结论 | 证据等级 | 说明 |
|---|---|---|
| 税前收入不能单独代表资本形成能力 | 强 | 三国资料和跨国统计框架共同支持 |
| 经常性预算余量是流动储备的前置条件 | 中强 | 美国SHED直接支持，三国需更多统一微观验证 |
| 净资产和流动性是不同维度 | 强 | SCF、HILDA/RBA及跨国wealthy hand-to-mouth研究支持 |
| 公共服务改变私人现金需求 | 强 | OECD调整后收入／实际消费框架支持 |
| 中国平均收支算术余量高于美国和澳大利亚储蓄率 | 不成立 | 指标来源、单位和处理不一致，不能排名 |
| 强制养老金一定等量提高家庭总储蓄 | 未定 | 需要识别其对其他储蓄和消费的替代程度 |
| 高储蓄必然意味着高经济选择权 | 不成立 | 预防性动机、资产锁定和固定负担可能使二者分离 |

## 8. 下一轮数据提取对象

第三轮为以下原型建立同口径表格：

1. 单身、无子女、进入劳动市场的青年；
2. 无子女双收入家庭；
3. 有未成年子女家庭。

每类再按以下状态分组：

- 租房；
- 有按揭自住房；
- 无按揭自住房。

核心输出为：

- 等价化现金可支配收入；
- 必要与固定支出；
- 经常性预算盈余；
- 流动缓冲月数；
- 养老金和住房等锁定资本；
- 金融资产与债务；
- 家庭外部支持。
